# platformio-node-proto
Docker image for PlatformIO + Node + protobuf

On [Docker hub](https://hub.docker.com/r/wala/platformio-node-proto/)


## Docker image for
* PlatformIO `3.3.0`
* Node `7.8.0`
* yarn `0.2.2`
* protobuf `3.2.0`
* protobuf python `3.2.0`
* nanopb `0.3.9-dev`
* protobufjs `6.7.3`
